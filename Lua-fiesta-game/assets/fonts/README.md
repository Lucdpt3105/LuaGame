# 📝 Hướng dẫn cài font tiếng Việt

## Cách 1: Dùng Noto Sans (Khuyên dùng)

1. Truy cập: https://fonts.google.com/noto/specimen/Noto+Sans
2. Nhấn "Download all" 
3. Giải nén file ZIP
4. Copy file `NotoSans-Regular.ttf` vào folder này (`assets/fonts/`)

## Cách 2: Dùng Arial (Nếu không muốn download)

Nếu hệ thống Windows đã có Arial, game sẽ tự dùng font mặc định hỗ trợ Việt.

## Cách 3: Các font khác hỗ trợ tiếng Việt

- DejaVu Sans (https://dejavu-fonts.github.io/)
- Roboto (https://fonts.google.com/specimen/Roboto)
- Tahoma (thường có sẵn trên Windows)

**Chỉ cần copy file .ttf vào folder này là được! 🎉**

## Kiểm tra

Chạy game và xem text Việt có hiển thị đúng không.
